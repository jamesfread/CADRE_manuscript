# CADRE proteomics project 
Repository for data and code related to the CADRE proteomics manuscript entitled "Multicohort analysis unveils axon guidance pathways linking small for gestational age to spirometric restriction". This repo is linked to the Dryad repository available at https://doi.org/10.5061/dryad.5mkkwh7h3.

This repo contains the Relative Fluoroscence Unit (RFU) values of the proteomic data, the phenotypic/clinical characteristics, and aptamer information files for cord blood and later life peripheral blood samples used for the study. Example QC code and analysis code are supplied as html Rmarkdowns. The sheep model scRNA-Seq data set is to large to be stored here, but can be accessed from the Dryad link above.
